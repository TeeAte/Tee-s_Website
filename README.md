# Cloudflare Worker 部署指南

## 功能清单

### 前台网站
- 首页：展示品牌、音乐播放器、最新3条动态
- 动态页：所有文章列表
- 相册页：图片画廊（自动从文章中提取带图片的内容）
- 关于页：个人介绍

### 后台管理（/admin）
- 安全登录：密码认证（默认密码：admin123）
- 文章管理：发布、编辑、删除动态
- 图片上传：支持点击图片区域上传
- 流量监控：访问量统计、折线图展示
- 访问记录：IP、设备、页面访问详情
- 系统设置：密码修改、站点信息配置

## 部署步骤

### 1. 创建 D1 数据库

在 Cloudflare 控制台操作：
1. 进入 **R2 & D1 数据库** > **D1 数据库**
2. 点击 **创建数据库**，命名为 `teeate-db`
3. 记录数据库 ID

### 2. 创建 KV 存储（用于图片）

1. 进入 **R2 & KV 存储** > **KV**
2. 点击 **添加命名空间**，命名为 `teeate-images`
3. 记录命名空间 ID

### 3. 绑定资源到 Worker

1. 进入 **Workers 和 Edge** > **Your Worker**
2. 找到 **设置** > **变量**
3. 添加绑定：

**D1 数据库绑定：**
- 变量名：`DB`
- 类型：`D1 数据库`
- 数据库：选择 `teeate-db`

**KV 存储绑定：**
- 变量名：`IMAGE_KV`
- 类型：`KV 命名空间`
- 命名空间：选择 `teeate-images`

### 4. 修改默认密码

在 worker.js 中找到第 119 行：
```javascript
if (password === 'admin123') {
```

将 `admin123` 改为你的密码。

### 5. 部署 Worker

1. 进入 **Workers 和 Edge** > **Workers**
2. 点击 **创建应用程序** > **使用代码创建**
3. 命名你的 Worker（如 `teeate-website`）
4. 点击 **编辑代码**
5. 将 `worker.js` 的内容全部复制粘贴进去
6. 点击 **保存并部署**

### 6. 配置路由（可选）

如果你想绑定自己的域名：
1. 在 Worker 设置中点击 **路由**
2. 添加你的域名，如 `teeate.com/*`

## 使用指南

### 前台访问
- 访问你的 Worker 域名即可看到网站

### 后台管理
- 访问 `/admin` 路径
- 输入密码登录
- 默认密码：`admin123`（记得修改！）

### 发布动态
1. 登录后进入"文章管理"标签
2. 填写标题和内容
3. 点击图片区域上传配图
4. 点击发布

### 查看统计
- 进入"流量统计"标签查看访问量
- 进入"访问记录"查看详细访问信息

## 注意事项

1. **图片存储**：当前使用 KV 存储图片，每个 KV 值最大 25MB，适合小图片
2. **数据库限制**：D1 是 SQLite，适合个人使用，日访问量建议在 10 万以内
3. **密码安全**：默认密码一定要修改！
4. **备份建议**：定期从 D1 控制台导出数据库备份

## 自定义内容

在 worker.js 中修改以下部分来自定义网站：

**前台副标题**（约第 468 行）：
```html
<div class="subtitle">很Niubi 的 Website</div>
```

**关于页面内容**（约第 516-521 行）：
```html
<div class="about-section">
  <h2>About Me</h2>
  <p>在这里写你的个人介绍...</p>
</div>
```
